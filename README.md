# HTML Fundamentals

A collection of HTML exercises completed as part of the Codédex HTML 
course, documenting my progression through core HTML concepts. Files 
progress from basic document structure and text formatting through to 
semantic elements, forms, and introductory CSS styling.

Each file contains comments documenting my understanding of the 
concepts practiced, written as I worked through each exercise.

## Included Files

### Foundations
- **elemental.html** — First exercise introducing basic body and paragraph elements
- **newspaper.html** — Heading hierarchy and line breaks
- **textformatting.html** — Bold, italic, underline and strikethrough text formatting
- **blueprint.html** — Standard HTML document boilerplate structure
- **lorem_ipsum.html** — Divs, classes, IDs and in-page anchor navigation

### Lists & Links
- **lists.html** — Ordered and unordered lists using a recipe as a real-world example
- **links_images.html** — Anchor elements, image embedding and telephone links
- **familytree.html** — Nested unordered lists representing a hierarchical structure

### Images & Media
- **biography.html** — Images as hyperlinks, alt text, and target="_blank"
- **band.html** — Combining headings, images, paragraphs and lists in a single page

### Layout & Styling
- **apartmentlisting.html** — Multiple elements combined into a structured page layout
- **my_top_8.html** — Internal CSS, classes, IDs, inline-block display and image styling
- **power_rangers.html** — CSS class and ID selectors, background colours and div sizing

### Forms
- **signup.html** — Basic form with text, email and password inputs
- **signup_v2.html** — Form validation with minlength, maxlength and required attributes
- **googlesearch.html** — Simple search form
- **contactus.html** — Contact form with textarea and label elements
- **rsvp.html** — Form with radio buttons and checkboxes
- **WELP.html** — Review form combining text input, radio buttons, number input and textarea with character limit

### Semantic HTML
- **blog.html** — Introduction to semantic structure with header and nav
- **blog2.html** — Added main and footer to complete core semantic layout
- **blog3.html** — Introduced section and article elements
- **blog4.html** — Added figure and figcaption for media content
- **blog5.html** — Final version with introductory CSS styling, horizontal rules and in-page anchor navigation

## Concepts Covered
- Document structure and boilerplate
- Headings, paragraphs and text formatting
- Links, images and media embedding
- Ordered and unordered lists, including nested lists
- Divs, classes and IDs
- Forms, inputs and validation
- Semantic HTML elements
- Introductory CSS styling

## How to View
- Clone or download the repository
- Open any .html file directly in your browser
- No dependencies or setup required
