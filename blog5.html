Stories


Python

Python was the first language which I sought to learn. It has an increasingly high demand with employers due to it's relationship to AI and automation, and hence, is a very valuable asset to acquire.

It also is one of the most beginner-friendly languages because of it's syntax similarities to the English language.

I proceeded through the course, learning the fundamentals of: variables, control flow, loops, lists, functions, classes & objects and modules. I utilised my knowledge of these concepts (and my own googling... lots of it) to create a data-persistent command line task manager!




HTML


HTML is a markup language which acts as the structure for all websites. It works in tandem with CSS and JavaScript to create beautiful webpages. Without HTML, CSS and JavaScript would not be able to execute their functions properly!

In itself, HTML isn't anything particularly exciting. It is, genuinely, the bare-bones of a website. In fact, this blog is written up in HTML!

I'm currently taking the HTML course on Codédex and this is part of an exercise. I'm being introduced to semantic elements such as articles and sections which makes the website more accessible for people with screen readers, for example. I'm excited to learn CSS and JavaScript to eventually add some beautiful, fun functionality to the page!




CSS


Like mentioned previously in the HTML article, CSS is a language for the web. It can change fonts, colours, spacings, etc. It takes HTML elements and manipulates them, making them look nicer.

I don't know too much about CSS, especially since I haven't began the course yet. What I do know, though, is that it is one of if not the most important languages for the web. I have already used some slight bits of CSS though, but how...?

I've implemented what is known as 'inline CSS'. This is CSS which lives directly in the HTML document rather than it's own seperate style sheet. Usually, this resides within the page 'head' so that the style can be referred to later in the document, however, it can also be directly linked to a single element if necessary.
